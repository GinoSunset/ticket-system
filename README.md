Проект "ticket-system" — это корпоративная система для управления заявками, складом и производственными задачами. Вот основные возможности и функции системы:

- **Управление заявками (тикетами):**
  - Создание, обновление, назначение исполнителей и отслеживание статусов заявок.
  - Возможность добавлять комментарии, файлы, акты выполненных работ.
  - Публичный доступ к заявке по ссылке для внешних пользователей.
  - Интеграция с электронной почтой для автоматического создания заявок из писем.
  - Поддержка кастомизации автоматического создания заявок (почта, API, Telegram).

- **Складской учет:**
  - Ведение учета компонентов и их типов.
  - Добавление, резервирование и списание компонентов.
  - Автоматическое резервирование компонентов под производственные задачи.
  - Управление доставками: создание, изменение дат, автоматическое перераспределение и резервирование компонентов при изменении статусов доставок.

- **Производство:**
  - Создание и отслеживание производственных задач.
  - Привязка номенклатур и компонентов к задачам производства.
  - Архивация компонентов после отгрузки.

- **Уведомления и интеграции:**
  - Email-уведомления исполнителям и менеджерам.
  - Интеграция с Telegram-ботом для оповещений.

- **Права доступа:**
  - Разграничение ролей пользователей (оператор, заказчик, исполнитель, администратор).

- **Технологии:**
  - Django, Celery, PostgreSQL, Redis, RabbitMQ, Docker, DRF
